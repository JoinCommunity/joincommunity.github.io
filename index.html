Streaming de video ao vivo na Globlo.com

A globo.com faz diversas transmissões de vídeos ao vivo, de partidas de futebol, noticiários a programas contínuos como o BBB.

Chegamos a transmitir mais de 100 sinais simultâneos e tivemos um pico de mais de 400 mil usuários concorrentes na Copa das Confederações de 2013.

O objetivo desta palestra é apresentar a arquitetura que torna a entrega do conteúdo e o monitoramento da plataforma, baseada largamente em software livre.

Discutir o papel das seguintes tecnologias livres utilizadas na globo.com para transmissão de vídeos ao vivo: GNU/Linux, Nginx, Módulo nginx push stream, Script NGINX com Lua, Redis, FFMPEG, Logstash, Graphite, Elastic Search, Kibana, Python e Ruby.

Leandro Moreira
Event Sourcing

Nessa palestra, apresento os conceitos fundamentais sobre Event sourcing, um importante método de persistência de entidades, ideal para cenários que demandam escalabilidade e eficiência.

Também apresento exemplos de implementação e encerro com considerações sobre os prós e contras da adoção.

Elemar Jr
Padrões para integração na Nuvem

A nuvem introduziu novos cenários para a "Cloud to OnPremises" e "Cloud to Cloud" integração entre sistemas. Nesta sessão, analisamos os padrões e as ferramentas disponíveis para integrar aplicativos, dados, identidade e redes entre esses ambientes.

Plataformas como o Windows Azure Service Bus tornam simples o compartilhamento da lógica de dados e de negócios do aplicativo, enquanto a Windows Azure Virtual Network pode fazer seus serviços de nuvem aparecem como serviços locais.

Serviços de Federação do Active Directory (AD FS) e Windows Azure Active Directory tornam possível a Federação de identidade.

Esta sessão inclui demonstrações destas tecnologias-chave e boas práticas para a integração em escala de nuvem.

Evilázaro Alves
Python para Zumbis: 12 mil alunos no primeiro MOOC brasileiro para ensino de programação

O que Python tem para atrair DOZE MIL inscritos num curso?!? Na data da submissão desta palestra ainda não paravam de se inscrever dezenas de novos zumbis todo dia.

Veremos os códigos que mais fizeram a alegria zumbi: baixar fotos de amigos do Facebook sem autenticação, fazer um mixador de áudio, hackear bibliotecas internas ou classes para obter a resposta fundamental sobre a Vida o Universo, e Tudo o Mais; onde 42 > 666, resolver a seletiva Hackaton Facebook em uma linha e muito mais.

Esta palestra é composta de 10 min de introdução e living code no resto do tempo.

Internet Of Things - Controlando E Monitorando Uma Residência Com Uso De Dotnet Microframework E Azure

Nesta apresentação falaremos sobre o uso de comunicação de dados entre maquinas (M2M) com o netduino controlando as luzes e o alarme de uma residência.

O Objetivo é apresentar de forma clara e prática o que precisamos de tecnologia para termos controle e monitoramento de uma residência.

Será também demonstrado o quanto é simples a integração do hardware em nossas vidas por meio das plataformas de desenvolvimento e nuvem da microsoft.

Será demonstrado um passo a passo do desenvolvimento de uma solução usando netduino como hardware que integrará a casa a núvem e o consumo do serviço via interface móvel.

O que será abordado:

  • Internet of Things
  • Conhecimentos básicos do Hardware necessário para a apresentação
  • MicroFramework dotNET
  • Segurança para a comunicação dos dados de controle
  • ASP.NET
  • MQTT
  • Real-time Web

Play! Framework - Produtividade extrema com Java Web Cansado de fazer as mesmas configurações toda vez que inicia um novo projeto Java? Cansado de sempre ter que integrar e encaixar os frameworks que utiliza em suas aplicações? Não quer mais brigar com seu framework para realizar tarefas simples? Conheça o Play! Framework e desenvolva aplicações Java para Web sem complicações e com altíssima produtividade. Conheça o Play! Framework e desenvolva aplicações Java para Web sem complicações. O Play! é um framework open source e mantido pela empresa TypeSafe. Escrito utilizando Scala, o mesmo tem suporte nativo ao desenvolvimento utilizando as linguagens Java ou Scala. Grandes empresas como o Linkedin, The Guardian e a Editora Abril já usam este framework que é altamente escalável, REST por padrão e baseado nos conceitos de COC (Convention Over Configuration) e KISS (Keep it simple, stupid). O Play! redefine a forma de desenvolver em Java para web pois não utiliza servlets e torna o HTTP uma interface simples, poderosa e flexível. Se você já conhece um pouco das facilidades e da produtividade do Rails vai perceber que o Play! traz isso e muito mais aos desenvolvedores Java. Frederico Maia Arantes
